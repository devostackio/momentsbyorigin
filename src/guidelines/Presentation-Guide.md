## Navigation Controls

### Basic Navigation
- **Left/Right Arrows:** Navigate between layouts (clickable or keyboard)
- **Dot Indicators:** Click any dot to jump to that specific layout
- **Eye Icon (Top Right):** Toggle navigation bar on/off for immersive viewing

### Presentation Modes

**Full Navigation Mode** (Default)
- Top bar shows current layout info
- Dot indicators visible
- Bottom instruction bar present
- Best for: Initial exploration, Q&A sessions

**Immersive Mode** (Click eye icon)
- All navigation elements hidden
- Full-screen presentation view
- Best for: Detailed layout review, investor pitches, focused discussions

## Recommended Presentation Flow

### 30-Minute Investor Pitch
1. **Layout 1** — Vision and mission (5 min)
2. **Layout 2-4** — Current ventures tour (10 min)
3. **Layout 5-7** — Future vision quick overview (8 min)
4. **Layout 8** — Investment opportunity (7 min)

### 15-Minute Quick Overview
1. **Layout 1** — Mission statement (3 min)
2. **Skip to Layout 8** — Investment tiers and opportunities (10 min)
3. **Return to specific layouts** — Based on investor interest (2 min)

### 60-Minute Deep Dive
1. **Layout 1** — Full vision presentation (8 min)
2. **Layouts 2-4** — Current ventures in depth (20 min)
3. **Layouts 5-7** — Future ventures detailed walkthrough (20 min)
4. **Layout 8** — Investment discussion and Q&A (12 min)

### Partner-Specific Presentations

**For Technology Partners:**
- Start with **Layout 3** (Stack by Origin)
- Jump to **Layout 8** (Partnership section)

**For Creative Partners (Artists, Chefs, Educators):**
- Start with **Layout 1** (Overview)
- Focus on **Layouts 2, 5, 6, 7** based on specialty
- End with **Layout 8** (Collective membership)

**For Community Organizations:**
- Start with **Layout 1** (Community impact)
- Focus on **Layouts 2, 7** (Media Labs, Origin × Studio)
- Discuss **Layout 8** (Futures Fund)

## Tips for Effective Presentation

### Do:
✓ Use immersive mode for detailed layout reviews
✓ Let investors navigate at their own pace during 1-on-1s
✓ Reference specific elements in each layout
✓ Connect dots between current and future ventures
✓ Emphasize community impact and mission
✓ Show how different investment tiers unlock different opportunities

### Don't:
✗ Rush through layouts — each tells a story
✗ Skip Layout 1 — it sets the foundation
✗ Forget to toggle back to navigation mode for Q&A
✗ Ignore the visual details — design quality matters
✗ Overlook the year indicators (Current vs. 2028)
